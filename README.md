# InacapSOSApp (Android)

Proyecto base de Android Studio para la app InacapSOS.

## Abrir en Android Studio

1. Abre Android Studio.
2. Ve a File > Open...
3. Selecciona la carpeta `InacapSOSApp`.
4. Espera a que Gradle sincronice el proyecto.

La URL base de la API se configura en:

`app/src/main/java/com/inacapsos/app/data/remote/ApiClient.kt`
